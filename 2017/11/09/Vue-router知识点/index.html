1、为何需要动态路由

我们经常需要把某种模式匹配到的所有路由，全都映射到同个组件。例如，我们有一个左导航组件，对于相应路由所有的子路由，都要使用这个组件来渲染。

3、id会被保存到this.$route.params

一个『路径参数』使用冒号 : 标记。当匹配到一个路由时，参数值会被设置到 this.$route.params，比如：路由：foo/5，this.$route.params.id==5

6、路由命名

有时候，通过一个名称来标识一个路由显得更方便一些，特别是在链接一个路由，或者是执行一些跳转的时候。你可以在创建 Router 实例的时候，在 routes 配置中给某个路由设置名称。

7、命名视图

有时候想同时（同级）展示多个视图，而不是嵌套展示，例如创建一个布局，有 sidebar（侧导航） 和 main（主内容） 两个视图，这个时候命名视图就派上用场了。
